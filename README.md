# Demography_AlcibarPerez/CalavazoCisneros

# Trabajo final Nuevo Leon

Este repositorio contiene los archivos utilizados para la construcción de tablas de vida, tasas de mortalidad, fecundidad y reproducción poblacional para Nuevo León.

## Carpetas

- data: bases de datos utilizadas.
- scripts: código R definitivo para los cálculos.
- output: archivos finales generados.
- images: gráficas finales.
- docs: informe final en PDF, diagrama de flujo y página web.

## Contenido

- Tablas de vida por sexo y año.
- Tabla de vida 2019 con causa eliminada: homicidios.
- Tasas Globales de Fecundidad.
- Tasas Brutas de Reproducción.
- Tasas Netas de Reproducción.
- Curvas TEFE comparativas para 2019.
- Demostración de la tasa de reemplazo de 2.1.
